Домашние задание к Уроку:

---------------------------------------------------------------------------------
Создать информационную систему по работе с сотудниками компании, студентами вуза

Операции по работе с базой данных

Структура БД (нарисовать SQL отношения между табл):
	id
	fio
	tel
	post
	department
	

Отдел организацииЖ
	Перевести в отдел 
	Удалить из отдела
	Проекты (задачи) отдела

Сортрудник:
1) Добавить
2) Удалить
3) Изменение данных по сотруднику
4) Пометка, что сотрудник уволен   


Файлы:
Выгрузить данные
Загрузить данные


Примерное меню действий системы:
================================================
    1: ['Вывести список отделов'],
    2: ['Вывести список всех сотрудников'],
    3: ['Вывести список сотрудников отдела],
    4: ['Добавить сотрудника'],
    5: ['Добавить отдел'],
    6: ['Переместить сотрудника в другой отдел'],
    7: ['Уволить сотрудника'],
    8: ['Выгрузить данные на диск'],
    9: ['Загрузить данные в БД'],
    0: ['Выход']
